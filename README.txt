Приложение React - chat разработано для общения коллег, в условиях ограниченного офлайн общения во время карантина для
поддержания коммуникации между сотрудниками, как по рабочим вопросам, так и для простого неформального общения и 
поднятия командного духа. Для этого предусмотрен вход под своим ником и паролем. По дефолту предлагается общение в рамках общего чата. Для обсуждения рабочих вопросов можно переключится на соответсвующий рабочий канал. Если имеющиеся каналы не устраивают - можно создать свой, нажав на кнопку с плюсом рядом со списком каналов - и указать 
свое название и описание канала. Каждый канал имеет свою историю общения. Если возникла необходимость что-то изменить в своих сообщениях - это
можно исправить сделав двойной клик по сообщению. Теперь нажав на карандаш - можно изменить его содержимое. Если нужно его удалить полностью - можно 
можно нажать на красный крестик и оно удалится. Чужие сообщения исправлять или удалять нельзя.
Это базовый функционал - в перспективе предполагается расширение функционала - смайлы, отправки файлов и многое другое...
Данные зарегистрированных пользователей, созданные каналы, а так же все сообщения записываются и хранятся в базе данных реального времени - Firebase.
Реализация UI - React, CSS стили - SCSS, бизнес менеджмент - redux . Deploy и hosting осуществлялся средствами Firebase.
В перспективе - перенос бинес логики и запросов на сервер  в специализированные
, возможно c использованием redux-thunk, мелкие доработки и реализация дополнительного функционала.
